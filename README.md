# 🎨 Frontend Web Basics

A collection of foundational web development projects focused on structural markup and styling. This repository serves as an archive of my early studies in building static user interfaces.

## 🎯 Objective

To understand the core building blocks of the web (HTML & CSS) before diving deep into backend architecture and software engineering. These projects demonstrate a practical understanding of semantic tags, layout structures, and visual design implementation.

## 🗂️ Projects Included

* **Landing-Page:** A clean, single-page promotional layout focusing on visual hierarchy.
* **Page-CVC:** A structural layout study inspired by a travel agency interface, practicing component positioning and image handling.
* **Recipe-Page:** A semantic HTML study presenting text hierarchy, ordered/unordered lists, and typography.
* **Class-Practices:** Assorted minor exercises covering HTML5 forms, varied inputs, and intermediate CSS selectors.

## 🛠️ Tech Stack

* **HTML5:** Semantic elements, document structure, and accessibility basics.
* **CSS3:** Flexbox, layout positioning, typography styling, and visual aesthetics.
